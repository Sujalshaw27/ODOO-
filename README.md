# ODOO - # 🚀 AssetFlow

> **Enterprise Asset & Resource Management System**

AssetFlow is a modern ERP platform that helps organizations manage assets, employees, departments, maintenance, and shared resources from a single dashboard. It replaces manual tracking with a secure, role-based, and scalable solution.

---

## ✨ Features

* 🔐 Secure Authentication & Role-Based Access
* 🏢 Department & Employee Management
* 📦 Asset Registration & Lifecycle Tracking
* 🔄 Asset Allocation & Transfer Workflow
* 📅 Shared Resource Booking with Conflict Detection
* 🔧 Maintenance Request & Approval Workflow
* 📋 Periodic Asset Audits & Discrepancy Reports
* 📊 Dashboard with KPIs & Analytics
* 🔔 Smart Notifications & Activity Logs

---

## 👥 User Roles

### 👑 Admin

* Manage departments & employees
* Create asset categories
* Assign user roles
* View organization-wide reports

### 📦 Asset Manager

* Register assets
* Allocate & transfer assets
* Approve maintenance requests
* Manage asset returns

### 🏢 Department Head

* View department assets
* Approve allocations & transfers
* Book shared resources

### 👨‍💼 Employee

* View assigned assets
* Book resources
* Raise maintenance requests
* Request transfers & returns

## 🔄 Asset Lifecycle

```text
Available
    ↓
Allocated
    ↓
Returned
    ↓
Available

OR

Available
    ↓
Under Maintenance
    ↓
Available

Other States:
Reserved • Lost • Retired • Disposed
```

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/AssetFlow.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
